# 🎮 Tic-Tac-Toe Game (C# Windows Forms)

A classic, fully functional Tic-Tac-Toe (X, O) desktop application built using **C# Windows Forms**. 

Beyond just the gameplay, this project was developed with a strong emphasis on clean code architecture, specifically focusing on **modular design** and **code reusability**. Instead of clustering all operations together, the game's logic is strictly separated into individual, single-purpose functions.

## ✨ Features

* **Classic Gameplay:** Play the standard 3x3 grid game of X and O.
* **Win/Draw Detection:** Accurately calculates winning combinations and tie games in real-time.
* **Game Reset:** Easily restart the match without closing the application.
* **Clean UI:** Simple, intuitive Windows Forms interface.

## 🧠 Architecture & Code Philosophy

A major focus of this project is its internal structure, prioritizing a "logic-first" approach:

* **Separation of Concerns:** Every specific task—such as validating a move, checking for a winner, switching turns, or updating the UI—is divided into its own isolated method.
* **Code Reusability:** By isolating tasks, the methods are highly reusable. Logic functions are kept separate from UI events, making the codebase easier to read, debug, and scale.
* **Maintainability:** The modular structure ensures that adding new features (like a score counter or an AI opponent) can be done with minimal disruption to the existing codebase.

## 🛠️ Technologies Used

* **Language:** C#
* **Framework:** .NET / Windows Forms
* **Paradigm:** Object-Oriented Programming (OOP)
